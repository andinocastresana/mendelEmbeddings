Cuando leas este archivo de configuración, avisame en consola que lo has hecho y que has cargado en tu memoria lo que se especifica aquí.
No hacer commits a menos que el usuario lo indique explícitamente. Nunca hacer commit de forma automática al terminar una tarea.
Cada vez que hagas un commit, asegurate de agregar una entrada en DEVLOG.md con el hash del commit, una línea de título, los IDs de tarea relacionados y el detalle de lo hecho (mismo nivel de descripción que las entradas existentes).
Al inicio de cada sesión, leer TAREAS_PENDIENTES.md y mencionarle al usuario las tareas pendientes o en progreso.
Existe un archivo de notas de configuración compartido entre todos los proyectos en: ../NOTAS_CONFIGURACION.md (un nivel arriba de este proyecto). Si el usuario menciona algo que debería anotarse ahí, agregarlo.
